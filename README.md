# web-AA
seminario: Los archivos que estan a continuacion constan tanto de backend como frontend.
para quien necesite saber como enlazar correctamente.
1. Se tiene que tener instalado previamente XAMPP (dentro del setup tiene que venir todo lo que ofrezca la app instalado)
2. los 8 archivos que estan en este repositorio se descargan y se deberan integrar en una carpeta llamada alianza_azul
3. esta carpeta se tendra que copiar dentro de la carpeta htdocs ( explorador de archivos>Este equipo>disco principal>xampp>htdocs )
4. ahora se debera iniciar xampp, darle start a "apache" y "mysql" y clickear "admin" en mysql
5. añadir la base de datos
6. en el localhost de phpmyadmin copiar lo siguiente en la url: "http://localhost/alianza_azul/"
7. confirmar que funciona
